✅ Redb Storage - Redb 임베디드 데이터베이스 - 단일 파일 트랜잭션
✅ File Storage - 파일 시스템 저장소 - 디렉토리 기반 테이블
✅ MongoDB Storage - MongoDB 저장소 - NoSQL 컬렉션에 SQL 인터페이스
✅ Redis Storage - Redis 저장소 - 키-값 캐시에 SQL 인터페이스
✅ Web Storage - 웹 저장소 - localStorage/sessionStorage 활용
✅ IndexedDB Storage - IndexedDB 저장소 - 브라우저 고급 로컬 DB
✅ Composite Storage - 복합 저장소 - 여러 저장소 통합 및 조인
✅ Git Storage - Git 저장소 - 버전 관리와 함께하는 데이터베이스
